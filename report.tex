\section{Introduction}
With the rise of messaging platforms like WhatsApp, group chats have become an important part of daily communication. These chats contain interesting patterns, such as who talks the most, how quickly people reply, and when they are most active. Studying these patterns can help us better understand group behavior and individual roles within conversations.

In this project, WhatsApp Group Wrapped, we build a system that analyzes and visualizes these patterns in a fun and interactive way, similar to Spotify Wrapped. First, a synthetic chat is generated using Python, where different users show realistic behaviors like varying activity levels and messaging styles. Then, the chat is analyzed to extract useful statistics about the group and its members.

The results are stored in a JSON file and used by a browser-based application to create a slideshow of insights. Each slide highlights a specific statistic, and individual profile pages show detailed information for each user. Overall, this project combines data generation, analysis, and visualization to turn chat data into meaningful and engaging insights.
